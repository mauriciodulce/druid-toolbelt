# druid-toolbelt

The basic productivity tools for OSX which every Druid developer needs.

## Tested on

- El Capitan

## How to install

Open terminal and run following command:
```
sh -c "$(curl -fsSL https://thisrepo/install.sh)"
```

## Tools included

- [Homebrew](http://brew.sh/)
- [Oh My ZSH!](http://ohmyz.sh/)
- [Ansible]()
- PHP
- Composer
- Drush
